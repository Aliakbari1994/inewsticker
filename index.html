Step 1 : add jquery library and inewsticker.js before closed body tag
Step 2 : add news ticker plugin after
$(document).ready(function() {
    $('#names').jtyping({
        speed  : 200,
        effect : 'slide',		
    });
});
Step 3 : change #name to your selector
Options
  • speed : text change speed
  • effect : text effect
  • dir : text direction ltr or rtl
  • typespeed : text typing speed
